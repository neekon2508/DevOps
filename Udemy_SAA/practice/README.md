# Certification Paths

## Architecture

- Solution Architect: SAA -> SAP -> Security
- Application Architect: SAA -> DA -> DEP + SAP

## Operations

- Systems Administrator: SysOps AA -> DevOps EP
- Cloud Engineer: SysOps AA -> Security -> DevOps EP + Advanced Networking

## DevOps

- Test Engineer: DA -> DevOps EP
- Cloud DevOps Enginner: Developer A -> SysOps AA + MLEA + DevOps EP
- DevSecOps Engineer: SysOps AA + MLEA + DevOps EP -> Security

## Security

- Cloud Security Engineer: SysOps AA -> Security + DevOps EP + Advanced Networking
- Cloud Security Architect: SAA + Security + SAP

## Development

- Software Development Engineer: Developer -> DevOps EP
- Network Engineer: SAA + Advanced Networking

## Data Analytics

- Cloud Data Engineer: SAA -> DEA -> MLEA + Security

## AI/ML

- ML Engineer: SAA -> MLEA -> DEA + Machine Learning
- Prompt Engineer: MLEA + ML
- ML Ops Engineer: SAA -> MLEA -> DEA + DevOps EP
- Data Scientist: SAA + MLEA -> ML
